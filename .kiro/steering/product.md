# Product Overview

NITechVacancyViewerは、名古屋工業大学の学生・教職員向けに教室および建物の空き状況をリアルタイムで表示するAndroidアプリケーションです。

## Core Capabilities

- **リアルタイム空き状況表示**: 教室と建物の現在の利用状況を可視化
- **教室予約機能**: 大学システムと連携した教室予約の閲覧・管理
- **認証連携**: 大学の認証システムとのシームレスな統合
- **オフライン対応**: ローカルキャッシュによる一時的なネットワーク切断時の動作保証

## Target Use Cases

- 学生が空き教室を探して自習場所を確保する
- 教職員が教室の利用状況を確認して会議室を予約する
- キャンパス内での移動前に建物の混雑状況を把握する

## Value Proposition

大学の既存Webシステムは操作が煩雑でモバイル最適化されていないため、本アプリはモバイルファーストのUIで素早いアクセスと直感的な操作を実現します。ローカルデータベースによるキャッシュ機構により、ネットワーク環境が不安定な建物内でも快適に利用できます。
